## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/yourusername/pixel-art-canvas.git
cd pixel-art-canvas
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Usage

1. **Select a Color** - Click on any color from the palette
2. **Draw** - Click and drag on the canvas to create pixel art
3. **Mobile Drawing** - Use touch gestures on mobile devices
4. **Clear Canvas** - Use the clear button to start over
5. **Download** - Save your creation as a PNG file
